Estructura básica de un script de JavaScript

Si lo estoy escribiendo en el HTML directamente, lo voy a encapsular en las etiquetas <script> y </script>.

Si lo estoy escribiendo en un archivo .js separado, lo voy a linkear a mi HTML mediante un link de src dentro las tags <script> y </script>. En estos .js externos no deben haber estas tags de script. Se pueden adjuntar diversos scripts externos, siempre en tags diferentes.

El separar el código de JS tiene algunas ventajas:

  1. Separa en documentos diferentes HTML y código
  2. Más fácil de mantener los documentos si se encuentran separados
  3. Páginas de JS en caché pueden acelerar la carga de ciertas webs

Output

Hay diversas formas de hacer output de un script de JS.

  1. Escribiendo en elementos HTML, usando innerHTML
  2. Escribiendo en el output del HTML con document.write()
  3. Escribiendo en una alert box con window.alert()
  4. Escribiendo a la consola del navegador con console.log()

Statements

El ; (punto y coma) separa statements en JS. Los statements en una línea están permitidos siempre que se separen con ;. Esta terminción no és obligatoria pero si muy altamente recomendada.

JS ignora los espacios en blanco, así que puedes anadirlos para hacer más comprensible el código. Es recomendable ponerlos alrededor de operaciones (= + - * /).

El largo recomendado de línea és de 80 caracteres.

Keywords

break: Terminates a switch or a loop

continue: Jumps out of a loop and starts at the top

debugger: Stops the execution of JavaScript, and calls (if available) the debugging function

do ... while: Executes a block of statements, and repeats the block, while a condition is true

for: Marks a block of statements to be executed, as long as a condition is true

function: Declares a function

if ... else: Marks a block of statements to be executed, depending on a condition

return: Exits a function

switch: Marks a block of statements to be executed, depending on different cases

try ... catch: Implements error handling to a block of statements

var: Declares a variable

Sintaxis

Hay dos tipos de variables, las fijas llamadas "literales" y las variables llamadas "variables".

Los números se escriben con o sin decimales, las strings son texto delimitado por "" o ''.

Las variables se utilizan para guardar valores de datos. JS usa el keyword "var" para declarar una variable. Un signo de igual (=) assigna valor a las variables.

JS es sensitivo a las mayúsculas y minúsculas, por lo que no es lo mismo "lastName" que "lastname". Los programadores ponen la primera letra en minúscula y todas las que irían después en mayúscula: firstName, lastName, masterCard, interCity.

Todas las variables deben estar identificadas con un nombre único.
